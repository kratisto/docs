## Ziel

Die [Web Cloud Databases](/links/web/databases) Lösungen können kostenlos über unsere [Performance Webhosting Angebote](/links/web/hosting) aktiviert werden. In diesem Fall werden sie dem Webhosting zugewiesen, von dem aus sie aktiviert wurden. Bei der Nutzung Ihrer Dienste kann es vorkommen, dass Sie Ihre Web Cloud Databases-Lösung vom Performance Webhosting abtrennen müssen, mit dem sie verbunden ist.

**Diese Anleitung erklärt, wie Sie so trennen Sie Ihre Web Cloud Databases Lösung von einem Webhosting.**

## Voraussetzungen

- Sie verfügen über eine [Web Cloud Databases Instanz](/links/web/databases), die mit einem [Performance Webhosting](/links/web/hosting) Angebot verbunden ist.
- Sie sind in Ihrem [OVHcloud Kundencenter](/links/manager) eingeloggt.
- Sie sind mindestens „[Administrator](/pages/account_and_service_management/account_information/managing_contacts)“ der Dienste, auf die Sie zugreifen möchten.

## In der praktischen Anwendung

Klicken Sie auf die Registerkarten unten, um die einzelnen **4** Schritte anzuzeigen.

> [!tabs]
> **Schritt 1**
>>
>> Loggen Sie sich mit Ihrem [OVHcloud Kundencenter](/links/manager) ein und gehen Sie dann in den Bereich `Web Cloud`{.action}.
>>
>> ![Web Cloud](/pages/assets/screens/control_panel/product-selection/web-cloud.png){.thumbnail}
>>
> **Schritt 2**
>>
>> Klicken Sie auf das Dropdown-Menü `Hosting-Pakete`{.action} und wählen Sie das betreffende Webhosting aus.
>>
>> ![Web Hosting](/pages/assets/screens/control_panel/product-selection/web-cloud/web-hosting/web-hosting-selection.png){.thumbnail}
>>
> **Schritt 3**
>>
>> Im Rahmen **Konfiguration** finden Sie den Vermerk **Web Cloud Databases**. Klicken Sie rechts auf den Button `...`{.action} und dann auf `Abtrennen`{.action}.
>>
>> ![Detach Web Cloud Databases](/pages/assets/screens/control_panel/product-selection/web-cloud/web-hosting/general-information/wcdb-detach.png){.thumbnail}
>>
> **Schritt 4**
>>
>> Wählen Sie auf der neu geöffneten Seite die gewünschte neue Abrechnung für Ihre Web Cloud Databases Lösung aus und folgen Sie den vorgegebenen Schritten bis zur Zahlung.
>>
>> ![Detach Web Cloud Databases](/pages/assets/screens/control_panel/product-selection/web-cloud/web-hosting/general-information/detach-my-web-cloud-databases.png){.thumbnail}
>>

Nach Abschluss der 4 Schritte wird Ihre Web Cloud Databases Lösung innerhalb weniger Stunden bereitgestellt.
