## Objetivo

Esta guía explica las versiones de **PHP**, **Python**, **Perl** y **Apache** utilizadas como parte de los servicios [Web Hosting](/links/web/hosting) a los que se puede acceder a través de la red de Web Hosting.

**Descubra los detalles de las versiones de PHP, Python, Perl y Apache.**

## Requisitos

- Tener contratado un [plan de hosting de OVHcloud](/links/web/hosting).
