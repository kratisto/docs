## Objetivo

As informações contidas neste guia dizem respeito às versões de **PHP**, **Python**, **Perl** e **Apache** utilizadas no âmbito dos serviços [Web Hosting](/links/web/hosting), acessíveis através da rede Web Hosting.

**Descubra os detalhes das versões PHP, Python, Perl e Apache.**

- Ter um serviço de [alojamento web OVHcloud](/links/web/hosting).
